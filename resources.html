links

date modified: 2023-08-20

i've made this page to document different resources i've been using to learn various types of development

the hope is that having these resources compiled might help someone else trying to learn how to do various things

webdev

  • htmlreference.io: every html attribute on one condensed page
  • cssreference.io: a spinoff of htmlreference.io for css attributes
  • javascript.info: a sleek, modern js tutorial for those new to the language

c++

  • w3school's c++ tutorial: contains pretty much every topic you'll need to start off with c++ programming

colors

  • terminal.sexy: good site for making 16 bit color schemes
  • colorcodes.io: a database of quality colors and their hexcodes

fonts

  • google fonts: fonts mostly licensed under ofl
  • font awesome: good for icons (cc 4.0 / SIL OFL 1.1, see license)
  • fontsquirrel: good amount of fonts in this one too, ofl mostly
